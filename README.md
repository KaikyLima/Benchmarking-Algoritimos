# Benchmarking-Algoritimos

Benchmark de algoritmos de ordenação implementado como requisito para a disciplina de Estrutura de Dados (PPGCC02), do curso de Mestrado da Universidade Tecnológica Federal do Paraná (UTFPR) - Campus Campo Mourão.

Para compilar, execute o comando ` gcc main.c src/*.c -o run_benchmark -lm ` no diretório do projeto. Em seguida, execute com o comando `./run_benchmark`.
